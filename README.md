# Ideas Board

A simple and intuitive app to write, organize, and manage your ideas.

## Features
- **Add new ideas**: Use the input field and click "Add Idea" button or press Enter
- **Persistent storage**: Ideas are automatically saved to localStorage and persist across page reloads
- **Delete ideas**: Remove individual ideas with the Delete button
- **Clear all**: Remove all ideas at once with the Clear All button
- **Empty state**: Friendly message when you have no ideas yet
- **Responsive design**: Beautiful gradient background with modern UI

## Usage
1. Open `index.html` in your browser
2. Type an idea in the input field
3. Click "Add Idea" button or press Enter to add it
4. Your ideas are automatically saved
5. Delete individual ideas with the Delete button
6. Use "Clear All Ideas" to remove everything (with confirmation)

## Technical Details
- Pure HTML, CSS, and JavaScript (no dependencies)
- Uses localStorage API for data persistence
- Fully functional offline
- Mobile-friendly responsive design
