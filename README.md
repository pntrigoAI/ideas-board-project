# Ideas Board

A simple app to write and organize ideas.

## Features
- Add new ideas using the input field.
- Display ideas in a list.
- Delete ideas from the list.

## Usage
1. Open `index.html` in your browser.
2. Type an idea and press "Enter" to add it.
3. Ideas appear in the list below; delete them with the "Delete" button.

This is a minimal app for brainstorming. Future enhancements can include:
- Edit ideas.
- Save ideas locally or on a server.
